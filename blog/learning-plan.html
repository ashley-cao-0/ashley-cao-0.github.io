My learning plan

Long term goal

My long term goal is to become a game developer. I love programming in general, so this may change if I find something else I prefer. There are some youtube videos on web games that I planned to watch.

Strengths and limitations

  • My strength is I learn quite fast if I'm interested in the topic. When that happens, I can give up all distraction and focus on learning that thing.
  • My weakness is dependence on motivation. When I try to learn something useful but not interested, I have a hard time starting to work on it.

Soft skills I want to learn

I want to learn to improve my self control and assertiveness. I want to reduce my media consumption, and be able to refuse requests and invitations, when there are things I need to finish.

Time management

I'll try to finish my assignments on time. I'll free up my weekend to make sure I have time if more work need to be done. I'll talk to my tutor if I find it too overwhelming.

Help seeking

I'll use discord to ask for help from the group, as well as message my tutor if needed.

What I expect from the Facilitation team

I expect to be supported, when I have questions and concerns. For the in-person classes, I hope it's easy to talk to someone (eg. a quick chat after class, a short catchup, etc.)

Other schedule
